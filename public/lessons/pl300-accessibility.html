Accessibility Basics

♿ What Is Accessibility?▼

Definition:

✓ Reports usable by everyone
✓ Including people with disabilities
✓ Without special modifications

Disability Types:

✓ Visual (blind, color blind)
✓ Hearing (deaf)
✓ Motor (mobility issues)
✓ Cognitive (learning disabilities)
🎯 Why Accessibility Matters▼

Benefits:

✓ Legal compliance (WCAG)
✓ Broader audience
✓ Better usability for all
✓ Professional responsibility
🌐 Web Standards (WCAG)▼

WCAG Levels:

✓ A: Minimal compliance
✓ AA: Recommended standard
✓ AAA: Enhanced (difficult)

Power BI Target:

✓ WCAG 2.1 Level AA

WCAG Standards

📋 WCAG Principles▼

Four Pillars:

✓ Perceivable: Users see/hear it
✓ Operable: Users can interact
✓ Understandable: Clear language
✓ Robust: Works with tech aids

Applied to Reports:

✓ Contrast (perceivable)
✓ Keyboard (operable)
✓ Labels (understandable)
✓ Semantics (robust)
🎨 Color Contrast (WCAG AA)▼

Requirements:

✓ Text: 4.5:1 ratio (normal)
✓ Text: 3:1 ratio (large, 18px+)
✓ Graphics: 3:1 ratio

Testing:

✓ Use contrast checkers
✓ Test on actual displays
✓ Avoid color-only coding
⌨️ Keyboard Navigation▼

Requirements:

✓ Tab through all controls
✓ Visible focus indicator
✓ Logical tab order

Power BI Support:

✓ Tab key navigation
✓ Enter/Space to activate
✓ Screen reader compatible

Design Principles

🎨 Color Use▼

Best Practices:

✓ Don't use color alone
✓ Add patterns/textures
✓ Use sufficient contrast
✓ Test color blindness simulation

Example:

✗ Red=bad, Green=good
✓ Red X=bad, Green ✓=good
📝 Text & Font▼

Recommendations:

✓ Minimum 12px font
✓ Sans-serif fonts (readable)
✓ Line spacing: 1.5x+
✓ Avoid all-caps text

Dyslexia-Friendly:

✓ Clear, simple language
✓ Short paragraphs
📋 Alt Text▼

What It Is:

✓ Description of visuals
✓ For screen readers
✓ Brief, meaningful

Example:

✗ "Chart 1"
✓ "Sales by region, Q4 2024"

Technical Implementation

⚙️ Alt Text in Power BI▼

Add Alt Text:

1. Select visual
2. Format → General
3. Alt text field
4. Describe visual clearly

Guidelines:

✓ Be specific
✓ Include key data
✓ Under 125 characters
🏷️ Labels & Titles▼

Requirements:

✓ Data source documented
