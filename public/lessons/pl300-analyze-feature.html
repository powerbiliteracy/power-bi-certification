Q&A Feature

Definition:

✓ Natural language interface
✓ Ask questions in English
✓ Auto-creates visualizations

Examples:

"What are top products?"
"Sales by region this year"
"Average price by category"

Steps:

1. Dashboard → Q&A box
2. Type question
3. Power BI interprets
4. Visual created

✓ Mention field names
