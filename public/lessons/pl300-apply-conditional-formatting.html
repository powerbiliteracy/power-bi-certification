Apply Conditional Formatting

Conditional formatting automatically applies colors, icons, and data bars based on cell values. This visual encoding helps viewers instantly identify patterns, outliers, and insights without reading numbers.

Format TypeUse CaseBest For
Color ScalesSequential valuesSales, temperature
Data BarsMagnitude comparisonRevenue, units
Icon SetsStatus indicatorsPerformance, risk
Highlight CellsSpecific valuesOutliers, targets
Custom RulesComplex logicMulti-condition rules
💡
Key Principle
Formatting should encode data, not decorate. Every color, bar, and icon should convey meaningful information.

Format Types

🎨 Color Scales▼

What They Are:

✓ Gradient from one color to another
✓ Low to High: Red to Green
✓ Diverging: Red to Green (with neutral)

When to Use:

Sales ranking: Light to dark
Temperature: Blue (cold) to Red (hot)
Performance: Red (poor) to Green (great)

Quick Setup:

Select cells → Home tab
Conditional Formatting → Color Scales
Choose 2-color or 3-color scale
📈 Data Bars▼

What They Are:

✓ Bar chart inside a cell
✓ Length represents value
✓ Easy magnitude comparison

When to Use:

Revenue amounts
Sales units sold
Performance metrics

Setup:

Select cells → Conditional Formatting
Data Bar → Solid Fill or Gradient
Choose color
🔶 Icon Sets▼

Types of Icons:

Traffic Lights: Red/Yellow/Green
Arrows: Up/Down/Right
Shapes: Circle/Diamond/Square
Ratings: Stars (1-5)

When to Use:

Status: Green = pass, Red = fail
Direction: Up = increase, Down = decrease
Performance tiers

Data Bars

📈 Creating Data Bars▼

Step-by-Step:

1. Select data range (e.g., sales values)
2. Home → Conditional Formatting
3. Data Bar → Solid Fill (or Gradient)
4. Choose color (blue, green, red)
5. Apply

Customization:

Fill: Solid, Gradient, or Transparent
Color: Any custom color
Minimum/Maximum: Auto or Fixed value
⚙️ Data Bar Settings▼

Important Options:

Show Value: Display numbers + bar
Hide Value: Bar only, numbers hidden
Value Axis: Show scale reference

Best Practices:

✓ Show value if exact numbers matter
✓ Hide value for cleaner look
✓ Use consistent colors across reports
💡 Data Bar Examples▼

Sales Performance:

Sales column: Blue data bars
Compare rep performance at a glance

Budget vs Actual:

Actual: Green bars
Over-budget: Red highlight + bars

Color Scales

🌈 3-Color Scales▼

Common 3-Color Combos:

Red-Yellow-Green: Bad-Neutral-Good
Blue-White-Red: Low-Medium-High
Green-Yellow-Red: Best-Mid-Worst

When to Use 3-Color:

Sales targets (below/at/above)
Performance ratings
Temperature ranges
🎨 2-Color Scales▼

Common 2-Color Combos:

White-Green: None to Maximum
Red-White: Minimum to Zero
Blue-Red: Low to High

When to Use 2-Color:

One-directional data (always increasing)
Revenue (white to green)
Loss (red to white)
⚙️ Configuring Scales▼

Setup:

Select range → Conditional Formatting
Color Scales → Choose preset or custom
Set Minimum, Midpoint, Maximum

Value Types:

Lowest/Highest: Auto from data
Number: Fixed value (e.g., 100)
Percentile: Based on distribution

Icon Sets

🔶 Icon Set Categories▼

Directional Icons:

Arrows: Up, Down, Right
Use: Trend direction

Status Icons:

Traffic Lights: Red, Yellow, Green
Use: Pass/Fail, Go/Stop

Shapes:

Circles, Diamonds, Squares
Use: Severity levels
📊 Applying Icon Sets▼

Steps:

1. Select range (numeric values)
2. Conditional Formatting → Icon Sets
3. Choose icon type (3, 4, 5 icons)
4. Configure thresholds
5. Apply

Example (Traffic Light):

Green: >= 80 (Good)
Yellow: 60-79 (Fair)
Red: < 60 (Poor)

Custom Rules

🎯 Highlight Cell Rules▼

Built-in Rules:

Greater Than: > $100,000
Less Than: < 50
Equals: Specific value match
Text Contains: "Error" or "Urgent"

Setup:

Select cells → Conditional Formatting
Highlight Cell Rules → Choose type
Set value and color
⚡ Top/Bottom Rules▼

Types:

Top 10%: Highest values
Bottom 10%: Lowest values
Top 10 Items: Top N values
Above Average: Values > mean

Use Cases:

Identify top performers
Flag underperformers
Outlier detection
🔧 Formula-Based Rules▼

Advanced Formatting:

Use formulas for complex logic
Example: =AND(A1>100, B1<50)
Apply multiple conditions

Setup:

Conditional Formatting → New Rule
Formula: =IF(condition, TRUE, FALSE)
Set format (color, font)

Scenario 1: Sales Performance

Icon: Green (above target), Red (below)

Best Practices

✓ Conditional Formatting Checklist▼
□ Format serves purpose (not decoration)
□ Colors accessible (color-blind safe)
□ Consistent across report
□ Not overwhelming (max 2-3 formats)
□ Scale/range appropriate
□ Works in black & white (if printed)
□ Clear interpretation (no guessing)
□ Performance: Not slowing report
⚠️ Common Mistakes▼
⚠ Too many formats - Keep it simple
⚠ Conflicting colors - Use one scheme
⚠ Inaccessible - Test for color-blind
⚠ Wrong scale - Review min/max values
⚠ Decorative only - Ensure it adds value
⚠ No legend - Explain what colors mean
⚠ Performance drag - Monitor impact
