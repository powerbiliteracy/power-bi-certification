Configure Automatic Page Refresh

Automatic page refresh updates visuals at specified intervals. You can set refresh to 1 second to 60 minutes, optimize for performance, and ensure reports display the latest data—enabling real-time monitoring dashboards and live data exploration.

Refresh TypeIntervalUse Case

Key Principle

Refresh Basics

🔄 What Is Auto Refresh?▼

Definition:

✓ Page reloads data automatically
✓ At fixed intervals
✓ User can still interact

Result:

✓ Always latest data
✓ No manual refresh needed
⏱️ Why Auto Refresh?▼

Benefits:

✓ Real-time monitoring
✓ Live dashboards
✓ Always current
✓ Reduced manual work
📍 When to Use▼

Good Candidates:

✓ KPI dashboards
✓ Operations monitoring
✓ Live sales dashboards
✓ System health pages

Not Needed:

✓ Static analysis

Configuring Refresh

🛠️ Setting Page Refresh▼

In Power BI Desktop:

1. View → Page refresh
2. Or: Format → Page
3. Turn on auto refresh
4. Set interval (30s - 60m)

In Power BI Service:

Edit mode → Page refresh
Same settings
⏱️ Refresh Intervals▼

Available Range:

✓ 30 seconds (minimum)
✓ 1, 5, 15 minutes
✓ 30, 60 minutes (max)

Choose Based On:

✓ Data change frequency
✓ Performance needs
✓ User expectations
🔧 Refresh Settings▼

Configuration:

Enable/disable toggle
Select interval
Apply to page

Scope:

✓ Per page setting
✓ Can vary by page
✓ Independent refresh
