Create Custom Tooltips

Custom tooltips display additional information when users hover over visuals. Unlike default tooltips, custom tooltips can be full report pages with visuals, filters, and complex layouts, enabling rich contextual exploration.

TypeContentComplexity
Default TooltipField values onlySimple
Custom TooltipReport page with visualsAdvanced
Named TooltipReusable tooltip pageFlexible
💡
Key Principle
Tooltips provide context without cluttering the main view. Use them to answer "why" and "what if" questions at a glance.

Tooltip Basics

💬 What Are Tooltips?▼

Definition:

✓ Pop-up information on hover
✓ Appears when user hovers over visual
✓ Disappears when mouse leaves

Default vs Custom:

✓ Default: Shows field values
✓ Custom: Shows rich content
🎯 Why Use Custom Tooltips?▼

Benefits:

✓ Explain underlying data
✓ Show related metrics
✓ Reduce cognitive load
✓ Enable drill-down exploration
✓ Context-aware insights

Use Cases:

✓ Performance explanations
✓ Trend analysis
✓ Benchmark comparisons
📍 Tooltip Behavior▼

Trigger:

✓ Hover over visual element
✓ Immediately displays

Context Passing:

✓ Data point filters tooltip
✓ Tooltip shows related data
✓ Dynamic based on selection

Tooltip Types

🏷️ Default Tooltips▼

Features:

✓ Automatically generated
✓ Shows field values
✓ Minimal configuration

Example:

Hover over bar: "Sales: $50,000"

Limitations:

✓ Text only, no visuals
✓ Limited customization
📄 Report Page Tooltips▼

Features:

✓ Full report page as content
✓ Include charts, tables, text
✓ Responsive to data context

Capabilities:

✓ Multiple visuals
✓ Filters auto-applied
✓ Rich formatting
🎨 Named Tooltips▼

Benefits:

✓ Create once, reuse many times
✓ Consistent across report
✓ Easy to update

Usage:

✓ Assign same tooltip to multiple visuals
✓ All visuals use same tooltip page

Creating Tooltips

✏️ Creating Custom Tooltips▼

Steps:

1. Create new blank report page
2. Design tooltip content (add visuals)
3. Set page as tooltip page
4. Assign to visual

Page Setup:

✓ Keep page small (fits tooltip)
✓ Remove page navigation if needed
✓ Design for quick scanning
📄 Tooltip Page Configuration▼

Mark as Tooltip:

1. Right-click page tab
2. Select "Page type" or "Is tooltip"
3. Confirm as tooltip page

Page Properties:

✓ Size: Small, square optimal
✓ Visuals: Limit to 2-3
✓ Filters: Auto-filter by context
🔗 Assigning to Visuals▼

Assign Tooltip:

1. Select visual
2. Format → Tooltip
3. Enable Tooltip → Select page
4. Done

Visual-Level Options:

✓ Show: On/Off
✓ Page: Select tooltip page
✓ Target fields: Which data points

Tooltip Pages

📄 Designing Tooltip Pages▼

Design Principles:

✓ Compact: Minimal scroll
✓ Focused: Answer one question
✓ Readable: Clear visuals

Layout Tips:

✓ One or two visuals max
✓ Title summarizing purpose
✓ Quick metrics/summary
🔄 Context Passing▼

Auto-Filtering:

✓ Tooltip receives context from hover
✓ Filters applied automatically
✓ Shows related data for that point

Example:

Hover "North Region" bar
Tooltip shows North data only
⚙️ Tooltip Page Settings▼

Hidden Pages:

✓ Tooltip pages can be hidden
✓ Right-click tab → Hide page
✓ Only accessible via hover

Naming:

✓ Name clearly: "Sales Tooltip"
✓ Indicates purpose

Formatting Tooltips

Styling Options:

✓ Font: Clear, readable
