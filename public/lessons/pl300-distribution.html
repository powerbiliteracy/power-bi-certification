Choose a Distribution Method

Distribution methods determine how users access content. You can share directly, publish apps, send via email, embed in websites, or export files—each with different use cases, permissions, and audiences.

MethodBest ForAudience Size
Direct ShareOne-to-one, small teamFew people
AppsCurated, self-serviceLarge audience
EmailQuick sharingSpecific people
EmbedWebsites, portalsExternal
ExportStatic filesOffline use
💡
Key Principle
Right method, right audience. Choose based on needs.

Direct Sharing

👥 What Is Direct Share?▼

Definition:

✓ Share directly with person/group
✓ Email invite sent
✓ Set permissions per recipient

Best For:

✓ Ad-hoc sharing
✓ Small groups
⚙️ How to Share▼

Steps:

1. Report/dashboard → Share
2. Enter email
3. Set permissions
4. Optional message
5. Share

Permissions:

✓ View
✓ Edit
✓ Share
⚠️ Direct Share Limits▼

Constraints:

✓ Manual per person
✓ Not scalable
✓ Hard to manage

Better For Large:

✓ Use apps instead
