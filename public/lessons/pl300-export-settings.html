Configure Export Settings

Export settings control how users can share and download data from reports. You can disable export entirely, allow specific formats (Excel, PDF, PowerPoint), or restrict export to the visual level—enabling controlled data sharing while protecting sensitive information.

SettingImpactUse Case
Export DisabledNo download optionSensitive dashboards
Visual Export OnlyExport current visualStandard reports
Full ExportExport underlying dataAnalysis tools
Format ControlLimit to Excel/PDFCompliance
💡
Key Principle
Export controls balance sharing with security. Choose granular controls for sensitive data.

Export Types

📊 Visual Export▼

What It Is:

✓ Export current visual only
✓ As image or data
✓ User right-clicks visual

Formats:

✓ PNG (image)
✓ CSV (data)
✓ Excel (data)
📄 Page Export▼

What It Is:

✓ Export entire page
✓ All visuals included
✓ From File menu

Formats:

✓ PDF (formatted)
✓ Excel (with formatting)
✓ PowerPoint (slide)
🔓 Underlying Data Export▼

What It Is:

✓ Export raw data
✓ All rows, not just visible
✓ For analysis/import

Restrictions:

✓ Can be disabled
✓ Requires permission
✓ RLS applied

Where to Configure:

Available Controls:

Real-World Scenarios

Scenario 1: Executive Dashboard

No data export allowed

Prevent data theft

Solution:

Users can view, not download
Only printing allowed
Scenario 2: Analyst Report

Requirement:

Excel export for analysis
Raw data needed
No PDF required

Solution:

Allow underlying data export
Excel format only
Restrict PDF/PowerPoint
Scenario 3: Mixed Sensitivity

Requirement:

Export allowed for some visuals
Sensitive charts locked
Granular control

Solution:

Visual-level settings
Summary charts: exportable
Detail charts: locked
Scenario 4: Presentation Export

Requirement:

PowerPoint for presentations
PDF for printing
No Excel data

Solution:

Allow PowerPoint export
Allow PDF export
Disable underlying data
