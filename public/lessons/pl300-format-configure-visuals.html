Format and Configure Visuals

Visual formatting transforms raw charts into professional, polished reports. Color choice, typography, legends, and axes configuration are critical for effective data communication.

ElementPurposeBest Practice
TitleDescribes chart contentClear, actionable statement
ColorsHighlight patternsConsistent, accessible palette
LegendIdentifies series/categoriesPositioned logically
AxesShows scale/valuesAppropriate intervals
TooltipShows exact valuesRelevant, formatted data
💡
Key Principle
Formatting is communication. Every color, font choice, label, and legend should serve a purpose.

Colors & Themes

🎨 Color Principles▼

Best Practices:

✓ Use 3-5 colors maximum
✓ Ensure color-blind accessibility
✓ Match colors to meaning
✓ Use consistent palette
✓ Test contrast for readability
🌈 Color Palette Strategy▼

Categorical:

Use distinct colors: Blue, Orange, Green, Red, Purple

Sequential:

Light to dark of same hue

Diverging:

Red ← Neutral → Green

Highlight:

Neutral background + accent color
🖌️ Theme Application▼

When to Apply:

✓ Corporate brand guidelines
✓ Report consistency
✓ Audience preferences

Titles & Labels

📝 Chart Titles▼

Good Title Formula:

Subject + What it shows + Period

Good vs Bad:

❌ Bad: "Data" or "Chart 1"
✓ Good: "Revenue Trends Show Q4 Surge"
🏷️ Axis Labels▼

Axis Labeling:

✓ X-axis: Clear category names
✓ Y-axis: Unit label with currency
✓ Format numbers (1M not 1000000)
📌 Subtitle & Annotation▼

Subtitle:

✓ Add context
✓ Explain anomalies
✓ Source

Annotations:

Highlight: Circle or arrow
Threshold: Line showing target
Period: Shade for special events

Legends & Tooltips

📖 Legend Configuration▼

When to Show:

✓ Multiple series
✓ Categories in pie/donut

Legend Position:

Right side: Most common
Bottom: Good for narrow charts
Inside corner: Space-saving
💬 Tooltip Configuration▼

What to Include:

✓ Category/Date
✓ Series name
✓ Value
✓ Percentage

Format Recommendations:

Currency: $1,250,000
Percentage: 35.2%
Large: 1.2M

Axes & Scale Configuration

📏 Axis Scale▼

Start from Zero?

✓ YES: Bar charts
✓ YES: Pie charts
✓ NO: Line charts
✓ NO: Scatter plots

Axis Intervals:

✓ Round numbers: 0, 100, 200
✓ Even spacing
✓ 4-6 grid lines maximum
🔢 Value Formatting▼

Number Formatting:

Large: $1M
Currency: $1,250
Percentage: 35% or 35.2%
Decimal: 1.5K, 2.3B
🔒 Fixed vs Dynamic Axes▼

Fixed Axis:

Use: Comparing multiple visuals
Benefit: Fair comparison

Dynamic Axis:

Use: Single visual
Benefit: Better detail
Risk: Exaggerate changes

Real-World Scenarios

Scenario 1: Executive Dashboard

Formatting Requirements:

✓ Color: Corporate blue + neutral
✓ Titles: Actionable insights
✓ Format: $1M, $2.5M
✓ Legends: Minimal
✓ Annotations: Target lines
Scenario 2: Analyst Report

Formatting Requirements:

✓ Value labels: Exact numbers
✓ Tooltip: Detailed + %
✓ Axes: All intervals
✓ Title: Specific
✓ Legend: All series
Scenario 3: Public Report

Formatting Requirements:

✓ Colors: High contrast, accessible
✓ Titles: Clear, engaging
✓ Labels: Minimal jargon
✓ Annotations: Key takeaways
✓ Scale: Start from zero
Scenario 4: Mobile Dashboard

Formatting Requirements:

✓ Legend: Bottom or hidden
✓ Title: Short
✓ Font: Large
✓ Labels: Show on tap
✓ Color: Good contrast

Best Practices

✓ Formatting Checklist▼
□ Title: Clear, actionable
□ Colors: 3-5 max, consistent
□ Legend: Positioned logically
□ Axes: Labeled, proper intervals
□ Tooltip: Relevant, formatted
□ Annotation: Highlights insights
□ Scale: Appropriate
□ Accessibility: Color-blind safe
⚠️ Common Mistakes▼
⚠ Vague title - BE SPECIFIC
⚠ Too many colors - USE 3-5 MAX
⚠ Legend covering data - POSITION OUTSIDE
⚠ Small labels - INCREASE SIZE
⚠ Non-zero bars - SHOW CONTEXT
⚠ Inconsistent format - STANDARDIZE
⚠ Low contrast - TEST
