Enable Personalized Visuals in a Report

Personalized visuals enable readers to customize their report experience. Users can save personal bookmarks, apply personal filters, and create custom views—all without affecting shared views. This creates tailored experiences for different audiences.

FeaturePurposeScope
BookmarksSaved visual statesShared by default
Personal BookmarksUser-only statesPrivate per user
Personal FiltersUser-specific dataPrivate per user
Personal ViewsCustomized layoutsPrivate per user
💡
Key Principle
One report, many views. Personalization lets each user see what matters to them.

Personalization Basics

✨ What Is Personalization?▼

Definition:

✓ User customizes their view
✓ Does not affect others
✓ Persists across sessions

Why Matter:

✓ Improve relevance
✓ Focus on key data
✓ Faster insights
👥 Personal vs Shared▼

Personal (Private):

✓ Only creator sees
✓ User-specific filters
✓ Doesn't affect others

Shared (Public):

✓ Everyone sees
✓ Designer-created
✓ Common views
⚙️ Enabling Personalization▼

Prerequisites:

✓ Report in Service
✓ Edit permission
✓ Feature enabled

Users Can:

✓ Create personal bookmarks
✓ Apply personal filters
✓ Save custom views

Bookmarks

📑 Shared Bookmarks▼

What They Are:

✓ Saved visual states
✓ Created by designer
✓ Available to all users

Create in Desktop:

1. Set up desired state
2. View → Bookmarks
3. New bookmark
4. Name and save
⭐ Personal Bookmarks▼

What They Are:

✓ User creates while viewing
✓ Private to that user
✓ Persist across sessions

Create in Service:

1. Set filters/selections
2. Bookmark icon (top)
3. Save as personal
4. Access from menu
🔄 Using Bookmarks▼

Navigation:

✓ Click bookmark name
✓ Report jumps to state
✓ All filters applied

Management:

✓ Rename bookmarks
✓ Delete old ones
✓ Pin favorites

Personal Views

👁️ What Are Personal Views?▼

Definition:

✓ Custom page layouts
✓ User rearranges elements
✓ Private to user

Capabilities:

✓ Reorder visuals
✓ Hide/show visuals
✓ Resize elements
🎨 Creating Personal Views▼

Enable in Desktop:

1. Format → Page
2. Personalization → Enable
3. Publish report

Users Can Then:

✓ Edit view (button appears)
✓ Rearrange/resize
✓ Save as personal
🔄 Managing Views▼

User Controls:
