Promote or Certify Power BI Content

Endorsement levels signal content quality and trustworthiness. You can promote content (owner endorsement) or certify it (admin verification), helping users discover reliable data sources and reports—building organizational trust in analytics.

LevelWho Can DoPurpose
NoneN/ANot endorsed
PromotedOwner/adminEndorsement
CertifiedAdmin onlyVerified quality
💡
Key Principle
Signal quality. Help users trust data. Build credibility.

Endorsement Basics

⭐ What Is Endorsement?▼

Definition:

✓ Quality signal for content
✓ Two levels: promoted/certified
✓ Helps discovery

Benefits:

✓ Users trust endorsed content
✓ Appears higher in search
✓ Reduces duplicate effort
🎯 Why Endorse Content?▼

Reasons:

✓ Ensure quality standards
✓ Prevent duplicate datasets
✓ Build single source of truth
✓ Guide users to best content

Promotion

📢 What Is Promotion?▼

Definition:

✓ Owner/admin endorsement
✓ "This is good"
✓ No formal verification

Use For:

✓ Recommended content
✓ Tested and working
✓ Owner confidence
⚙️ How to Promote▼

Steps:

1. Dataset/report details
2. Click three dots menu
3. Select "Promote"
4. Confirm

Requirements:

✓ Must be owner/admin
✓ In workspace

Certification

✓ What Is Certification?▼

Definition:

✓ Admin formal verification
✓ "Official trusted source"
✓ Highest endorsement

Requirements:

✓ Admin only can certify
✓ Usually after review
⚙️ How to Certify▼

Process:

1. Admin reviews content
2. Item details page
3. Certify option
4. Add message
5. Confirm

Message:

✓ Explain why certified

Promotion vs Certification

📊 Key Differences▼

Promotion:

✓ Owner/admin action
✓ Quick endorsement
✓ "I recommend this"

Certification:

✓ Admin only
✓ Formal verification
✓ "This is official"
🎯 When to Use▼

Promote When:

✓ Content is good
✓ Owner confident

Certify When:

✓ Official data source
✓ After thorough review
