Group and Layer Visuals Using Selection Pane

The Selection pane provides a master list of all report elements. You can organize visuals into groups, control layering (z-order), show/hide elements, and manage complex page layouts—all without clicking individual elements on the canvas.

Grouping Visuals

📦 Creating Groups▼

Grouping Method:

1. Select multiple visuals
2. Right-click → Group
3. Elements grouped in pane
4. Can move as one unit

Multiple Selection:

✓ Ctrl+click visuals on canvas
✓ Or Ctrl+click in pane
✓ Then right-click → Group
🎨 Working with Groups▼

Benefits:

✓ Move group as one
✓ Resize group together
✓ Hide entire group

Group Organization:

Layering & Z-Order

↕️ Understanding Z-Order▼

What Is Z-Order:

✓ Visual layering depth
✓ Front to back ordering
✓ Controls overlap appearance

In Selection Pane:

✓ Order in list = layer order
✓ Top = Front layer
✓ Bottom = Back layer
⬆️⬇️ Reordering Layers▼

Change Layer Order:

1. Right-click element in pane
2. Arrange → Send Forward/Back
3. Or drag in pane to reorder

Options:

✓ Bring to Front
✓ Send to Back
✓ Move Forward/Backward
🎯 Overlapping Elements▼

Controlling Overlap:

✓ Background shapes behind visuals
✓ Icons on top of data
✓ Text boxes over visuals

Use Cases:

✓ Design backgrounds
✓ Layered annotations
✓ Visual emphasis

Best Practices
