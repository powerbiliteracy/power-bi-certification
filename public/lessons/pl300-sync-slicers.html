Configure Sync Slicers

Sync slicers synchronize slicer selections across multiple pages. When configured, selecting a value in one slicer automatically applies that filter to other pages, creating a unified filtering experience without duplicating slicers on every page.

ScenarioSetupResult
One slicer, one pageStandardFilters that page only
Duplicated slicersManualRedundant, confusing
Synced slicersConfiguredUnified experience

Key Principle
One slicer, multiple pages. Sync makes navigation seamless and reduces user confusion.

Slicer Sync Basics

🔄 What Is Slicer Sync?▼

Definition:

✓ Synchronize slicer across pages
✓ One slicer affects multiple pages
✓ Selections persist across navigation

Key Benefit:

✓ Users don't re-select on each page
🎯 Why Use Sync Slicers?▼

Benefits:

✓ Consistent filtering across pages
✓ Reduce slicer redundancy
✓ Improve user experience
✓ Cleaner report layout
✓ Maintain selection context
📍 Default Behavior▼

Without Sync:

✓ Each slicer filters current page
✓ Other pages unaffected
✓ Users must re-select on each page

With Sync:

✓ Selection persists across pages
✓ All synced slicers show same state

Sync Types

🔄 Source Slicer▼

What It Is:

✓ Master slicer on primary page
✓ User interacts with this one
✓ Synchronizes to other pages

Use Cases:

✓ Home/Overview page slicer
✓ Common filter everyone uses
📄 Target Pages▼

What They Are:

✓ Other pages receiving sync
✓ Can have own slicer (optional)
✓ Or just receive the filter

Options:

✓ Show slicer on target (visible sync)
✓ Hide slicer (invisible sync)
🎨 Visible vs Hidden Sync▼

Visible Sync:

✓ Slicer visible on page
✓ Shows current selection
✓ User sees synchronization

Hidden Sync:

✓ Filter applied, no slicer shown
✓ Cleaner page layout
✓ Must have source slicer visible

Configuring Sync

⚙️ Setting Up Sync Slicers▼

Steps:

1. Create slicer on source page
2. Right-click slicer → Sync slicer
3. Panel opens: Slicer options
4. Select: Which pages to sync to
5. Choose: Visible or hidden on target

Result:

✓ Synchronization configured
✓ Test by navigating
📋 Sync Pane Configuration▼

Configure Options:

1. Right-click slicer → Sync slicer
2. Pane shows all pages
3. Toggle: Sync on/off per page
4. Toggle: Show on this page

Page Options:

✓ Current page: Source slicer
✓ Other pages: Sync on/off
✓ Show slicer: Visible/hidden
🔀 Multiple Sync Slicers▼

Setup:

✓ Create multiple slicers
✓ Sync each independently
✓ Each filters different field

Example:

Slicer 1: Region (synced)
Slicer 2: Year (synced)
Both affect all pages
