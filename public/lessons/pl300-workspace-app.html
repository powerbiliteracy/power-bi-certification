Configure and Update a Workspace App

Apps are curated packages of workspace content distributed to users. You can create apps, configure settings and permissions, update content while maintaining versions, and manage user access—enabling controlled distribution of reports and dashboards.

StageActionAudience
CreateSelect content for appWorkspace admin
ConfigureSet display, nav, permissionsApp owner
PublishRelease to viewersOrganization
UpdateModify and republishDeployed users
💡
Key Principle
Apps curate and distribute. Control what users see.

Updating Apps

🔄 Updating Process▼

Steps:

1. Make changes in workspace
2. Go to app settings
3. Click "Update app"
4. Publish

Notes:

✓ No version control
✓ Users see latest
✓ Immediate deployment
📝 What Can Change▼

Update Content:

✓ Report data/queries
✓ Dashboard tiles
✓ Descriptions

Update Settings:

✓ App name/description
✓ Logo/branding
✓ Navigation structure
⚠️ Update Considerations▼

Be Aware:

✓ Changes affect all users
✓ Immediate deployment
✓ No rollback option

Best Practice:

✓ Test first in workspace
✓ Announce changes
✓ Schedule updates
